# Rock Paper Scissors Game ✊📄✂️

## Project Description

Rock Paper Scissors Game is a simple console-based C++ application where a player competes against the computer. The computer generates a random choice, and the winner is determined based on the classic game rules.

This project helps beginners understand random number generation, conditional statements, loops, and user interaction in C++.

## Features

- Player vs Computer gameplay
- Random computer choice generation
- Winner determination logic
- User-friendly console interface
- Multiple rounds support (optional)

## Technologies Used

- C++
- Standard Library

## Concepts Covered

- Variables and Data Types
- Conditional Statements (if-else)
- Loops
- Functions
- Random Number Generation
- User Input and Output

## How to Run

1. Clone or download this repository.
2. Open the project in any C++ compiler or IDE.
3. Compile and run the source file.

Example:

```bash
g++ RockPaperScissors.cpp -o RockPaperScissors
./RockPaperScissors
```
